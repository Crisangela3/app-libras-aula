# OuçaPorMim 🎙️

**OuçaPorMim** é uma solução de acessibilidade digital que transforma áudio em texto em tempo real. O projeto foi idealizado para apoiar a inclusão escolar, permitindo que estudantes com deficiência auditiva acompanhem aulas, palestras e reuniões com autonomia.

## 🚀 Funcionalidades

- **Transcrição em Tempo Real**: Reconhecimento de voz instantâneo com suporte a termos acadêmicos.
- **Integração com Aulas Online**: Botão dedicado para capturar o áudio diretamente do Google Meet.
- **Gestão de Conteúdo**: Salva transcrições em um histórico organizado por data e categoria (Aulas Antigas ou Pasta Médica).
- **Exportação de Dados**: Geração de arquivos PDF para estudo e revisão posterior.
- **Interface Intuitiva**: Design focado na usabilidade, utilizando uma paleta de cores acessível (Ciano, Gold e Bege).

## 🛠️ Tecnologias Utilizadas

- **React.js**: Para uma interface dinâmica e responsiva.
- **Web Speech API**: Motor de reconhecimento de voz nativo do navegador.
- **Lucide React**: Ícones modernos para navegação.
- **jsPDF**: Biblioteca para conversão de texto em documentos PDF.

## 📖 Como Usar

1.  Abra a aplicação e clique em **AJUDA** para entender os comandos.
2.  Use o **Microfone Central** para capturar áudio ambiente.
3.  Utilize o botão **Conectar Áudio do Meet** para transcrição em chamadas de vídeo.
4.  Salve suas anotações para que sejam movidas automaticamente para o histórico após 7 dias.
